# 3D Air-Jet Sorting Simulator

This repository contains a Jupyter Notebook implementation of a 3D rigid-body air-jet sorting simulator for CBE499 Project 3.

## Main features

- 3D rigid-body dynamics
- Surface point cloud representation
- Quaternion-based orientation update
- Gaussian air-jet field
- Local surface force and torque calculation
- Impulse-based detachment
- x-pass-region sorting criterion
- Interactive Jupyter widgets
- 3D trajectory animation

## How to run

Open `KAIST_2026S_CBE499_Project3.ipynb` in Jupyter Notebook or JupyterLab.

If the animation window turns on and then disappears after clicking the Animate 3D button, please press it again.
